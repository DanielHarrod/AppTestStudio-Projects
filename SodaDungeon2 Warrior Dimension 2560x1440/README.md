## Soda Dungeon 2 - Warrior Dimension - Fully Automated Gameplay

Projects created with [AppTestStudio](https://github.com/DanielHarrod/AppTestStudio).  

Download into your "Documents\App Test Studio" a folder

Start->Run: %USERPROFILE%\\Documents\\App Test Studio

C:\Users\(Current User)\Documents\App Test Studio\SodaDungeon2 Warrior Dimension 2560x1440\

## System Setup

### Monitor Settings
This project requires the monitor resolution to be set to 2560x1440, 100% monitor scaling.
**Important:** Monitor scaling is not supported. Set scaling to **100%**.

### Game Settings
Soda Dungeon 2 to use Display 2560x1440, Full Screen.
Have made it to the Warrior Dimension and have 1 loadout slot enabled.

### What's it do?
- Effectively it plays the game for you indefinately.
- Goes to the bar when there is no party selected.
- Loads a party from loadout.
- Goes into the dungeon.
- Clicks on Warp or Healing Fairy.
- When the party is defeated it starts a new party and continues.
